# Arkanoid Demo

## How to setup?

1. `$ git clone https://github.com/EmanuelDubor/scala-vainilla.git`
2. `$ git clone https://github.com/EmanuelDubor/scala-vainilla-demo-arkanoid.git`

#### How to run?

1. `$ cd scala-vainilla-demo-arkanoid/desktop`
2. `$ gradle run`

#### How to deploy?

1. `$ cd scala-vainilla-demo-arkanoid/desktop`
2. `$ gradle dist`
